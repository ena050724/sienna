Return to The Ocean

Case Studies

A selection of experimental web projects exploring minimal layout, interactive reading systems, and playful useless interfaces.

tdingsun - Aayushi

This website uses a minimal layout with diverse interaction methods. Text can scroll vertically inside small blocks and slide horizontally, creating a reading experience similar to Japanese calligraphy panels, which makes navigation feel intentional and engaging.

JET SET WILLY Variations ©1984

The website using a strong yellow background and tightly arranged images. The compressed composition creates the feeling of a fragmented visual narrative, almost like reading a story through sequential screenshots.

Paint Toys – Oil

The title is striking and the site presents a collection of playful Japanese-style mini games such as stone stacking, pixel drawing, and scratch art. The interface encouraging creativity and exploration.

Jeffrey Deitch

https://deitch.com

This website feels visually rich and dynamic.A big animation in the homepage make it immediately feel alive. When clicked, each image opens into more visual content, so the site feels like a gallery that is already in motion before you even enter.

Art.Teleportacia

https://art.teleportacia.org

I really like the idea behind this site. It starts from a tiny home button, but clicking it opens up endless little worlds. It feels intimate and exploratory, almost like reading someone’s diary through hyperlinks and discovering hidden thoughts page by page.

Simone Hoang

https://simonehoang.com

This website is striking because each image starts blurred and only becomes visible when the cursor hovers over it. The left bar and top bar also act automatically just like curtain, which makes the whole site feel controlled and aesthetic. The interaction is simple, but very memorable.
